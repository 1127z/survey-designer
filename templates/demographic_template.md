# 人口学变量模板

## 使用说明

人口学变量放在问卷末尾（致谢页之前）。根据研究对象选择需要的变量，不要全部照搬。

## 大学生群体常用人口学变量

### 基本信息类

| 变量 | 题干 | 选项 | 备注 |
|------|------|------|------|
| 性别 | 你的性别是？ | 男 / 女 / 不愿透露 | 建议加"不愿透露"选项 |
| 年龄 | 你的年龄是？ | ____岁 | 填空题，或给范围选项 |
| 年级 | 你的年级是？ | 大一 / 大二 / 大三 / 大四 / 研究生及以上 | 根据学校类型调整 |
| 专业类别 | 你的专业属于？ | 文科 / 理科 / 工科 / 医科 / 艺术类 / 其他 | 可改为更细的专业分类 |
| 政治面貌 | 你的政治面貌是？ | 中共党员(含预备) / 共青团员 / 民主党派 / 群众 | 部分研究需要 |
| 生源地 | 你上大学前的常住地是？ | 省会/直辖市 / 地级市 / 县级市/县城 / 乡镇 / 农村 | 注意选项不要用"城市/农村"二分法 |

### 经济状况类

| 变量 | 题干 | 选项 | 备注 |
|------|------|------|------|
| 月生活费 | 你每月的生活费大约是？ | 1000元以下 / 1000-1500元 / 1501-2000元 / 2001-3000元 / 3000元以上 | 敏感题，放末尾 |
| 家庭月收入 | 你的家庭月均收入大约是？ | 不愿透露 / 5000元以下 / 5000-10000元 / ... | 高度敏感，加"不愿透露" |

### 行为习惯类（根据研究主题选配）

| 变量 | 题干 | 选项 | 适用研究 |
|------|------|------|---------|
| 网购频率 | 你平均每月网购几次？ | 0次 / 1-2次 / 3-5次 / 6-10次 / 10次以上 | 消费行为研究 |
| 社交媒体使用 | 你每天使用社交媒体的时间大约是？ | 不到1小时 / 1-2小时 / 2-4小时 / 4小时以上 | 社交/心理健康研究 |
| AI工具使用 | 你使用AI工具（如ChatGPT）的频率是？ | 从未使用 / 偶尔使用 / 每周几次 / 每天使用 | 技术接受研究 |

## 编写规则

1. 月收入/家庭收入等敏感经济题必须加"不愿透露"选项
2. 年龄建议用填空题（选项化会损失精度）
3. 生源地不要用"城市/农村"二分法——太粗糙且带有价值判断，应提供多层选项
4. 人口学变量题项编号以 D 开头（D1, D2, D3...），与核心题项区分
5. 不要问与研究无关的人口学变量（如果研究不涉及家庭收入，就不要问）
6. 选项的粒度要匹配研究对象（如生活费选项应参考当地大学生实际消费水平）
