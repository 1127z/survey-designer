# 筛选题模板

## 什么是筛选题？

筛选题（Screening Questions）放在问卷最前面（封面语之后），用于确认受访者是否符合研究对象条件。不符合条件的受访者应跳转至致谢页，避免无效作答。

## 使用场景

- 研究对象限定特定群体（如"在校大学生""AI工具使用者""有网购经历的人"）
- 需要排除不符合条件的答卷
- 多版本问卷需要分流（如"使用者版本"vs"非使用者版本"）

## 常见筛选题类型

### 1. 身份筛选

确认受访者是否属于目标群体。

```
S1. 你目前是否为在校大学生？
A. 是 → 继续
B. 否 → 结束问卷
```

### 2. 经历筛选

确认受访者是否具备研究所需的特定经历。

```
S2. 你是否使用过 ChatGPT、Kimi、豆包、文心一言等 AI 工具？
A. 是 → 继续使用者版题目
B. 否 → 跳转至非使用者版题目 / 结束问卷
```

### 3. 频率筛选

确认受访者的行为频率是否达到研究门槛。

```
S3. 过去三个月内，你是否有过因心情变化而产生的消费行为？
A. 是 → 继续
B. 否 → 结束问卷
```

### 4. 年龄/年级筛选

确认受访者属于特定年龄段或年级。

```
S4. 你的年级是？
A. 大一
B. 大二
C. 大三
D. 大四
E. 研究生及以上
F. 其他 → 结束问卷
```

## 编写规则

1. 筛选题编号以 S 开头（S1, S2, S3...），与核心题项区分
2. 不符合条件的选项必须标注跳转方向（→ 结束问卷 / → 跳转至第X题）
3. 筛选题不超过 3 道，过多会导致受访者流失
4. 筛选题放在封面语之后、热身题之前
5. 如果研究对象没有特殊条件（如"全体大学生"），可以不设筛选题
