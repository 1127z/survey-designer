# survey-designer

社会科学学术研究问卷编制技能。

## 功能

从研究信息采集到问卷定稿，线性流程逐步确认，帮你产出一份可直接使用的学术研究问卷。

## 适用场景

- 本科/研究生课程作业中的问卷设计
- 社科类学术研究（社会学、心理学、教育学、公共管理等）
- 调研大赛申报材料中的问卷部分

## 不适用场景

- 市场调研 / 用户反馈等非学术问卷
- 需要完整数据分析方案的研究设计
- 伦理审查申请

## 流程

1. **信息采集** — 收集研究主题、对象、问题、目的等 9 项前置信息
2. **概念化** — 拆解构念→维度→指标，输出映射表
3. **量表策略** — 优先匹配成熟量表，标注采用/改编/自编
4. **题项编写** — 按维度出题，遵循 Fowler 五条铁律
5. **编排定稿** — 封面语→热身题→核心题→人口学变量→致谢

## 使用方法

直接说"帮我做一份关于xxx的问卷"即可触发。
