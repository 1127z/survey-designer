# 量表选择规则

## 核心原则

1. **能用成熟量表就别自己编** — 成熟量表已经过信效度验证，学术认可度高
2. **不能直接采用的，先考虑改编** — 改编比自编省力，且有原量表做参照
3. **只有成熟量表完全无法覆盖时才自编** — 自编题项必须经过预测试验证信效度

## 选择流程

```
该维度是否有成熟量表可用？
  │
  ├─ 是 → 该量表是否有已验证的中文版？
  │         │
  │         ├─ 是 → 直接采用中文版（需核对验证文献）
  │         │
  │         └─ 否 → 需要翻译+验证，或基于原量表维度自编中文题项
  │
  └─ 否 → 该领域是否有综述/理论框架可参考？
            │
            ├─ 是 → 基于理论框架自编题项，标注"参考[文献]自编"
            │
            └─ 否 → 从研究问题出发自编，标注"自编"，需在预测试中重点验证
```

## "直接采用"的判定标准

满足以下**全部条件**才能标注为"直接采用"：

1. 量表原文已公开发表在同行评审期刊上
2. 已有中文版翻译并在中文样本中验证过信效度
3. 量表的适用对象与研究样本匹配（如原量表针对成人，不直接用于青少年）
4. 量表不涉及商业版权限制（部分量表仅限学术研究免费使用）

**如果只满足第1条但不满足第2-4条，应标注为"改编"而非"直接采用"。**

## "改编"的适用场景

1. 原量表语言/文化背景与目标人群不匹配（如英文量表用于中国大学生）
2. 原量表语境与研究方向不同（如冲动购买量表调整为情绪驱动消费语境）
3. 原量表题量过大，需要精简（如从36题精选10题）
4. 原量表的某些维度不需要，只保留部分维度

改编时必须：
- 在论文中注明改编理由和具体调整内容
- 在量表来源清单中标注"改编自[量表名]，调整了[具体内容]"
- 改编后的量表需重新检验信效度

## "自编"的适用场景

1. 研究领域尚无成熟量表（如"补偿性消费"）
2. 研究概念较新，现有量表无法覆盖
3. 研究需要测量特定文化背景下的现象

自编时必须：
- 标注"自编"或"参考[文献]维度自编"
- 说明为什么没有成熟量表可用
- 在预测试中重点验证信效度
- 提供构念-维度-指标的完整映射逻辑

## 不同研究类型的量表策略倾向

| 研究类型 | 量表策略倾向 | 原因 |
|---------|-------------|------|
| 描述现状型 | 可多用自编行为频率题 | 重点在描述，不需要严格的心理测量 |
| 差异比较型 | 优先成熟量表 | 需要跨组可比的标准化测量 |
| 影响因素分析型 | 自变量+因变量均需成熟量表 | 需要可靠的变量测量才能做回归 |
| 中介/调节机制验证型 | 所有变量必须用成熟量表 | 机制验证对测量精度要求最高 |
| 需求/满意度调查型 | 满意度部分可自编，需配合开放题 | 满意度量表结构相对简单 |
| 政策建议型 | 可混合使用 | 侧重政策相关性，测量精度要求适中 |
